This minimap is based on the movie "Gone in 60 seconds".

Before installation make backup of those files:

r.gxt		(optional)
e.gxt
test1.seq
directory FRONTEND/

For install "Gone in 60 seconds" just copy catalog DATA from archive to catalog \DATA in directory with game.
